Count Battle!

This is a fork of Anton Tobi's Count! with some minor updates.

For each position, count which side is winning, assuming no komi and no prisoners. All positions have the same number of black and white stones, and there are no sekis, so area scoring and territory scoring will give the same result.

The positions are manually generated, so there's a relatively small number of them (100 at the moment), which are presented at random orientations. To avoid tempting you to memorize specific positions, for many of them I've included a very similar position where the result is the opposite.

When you load the page, you are prompted to choose a time limit per board in seconds. Leave the field empty to play without timer.

The game continues until you run out of time or give an incorrect answer. Set your own personal goal (for instance "20 boards in 20 seconds each") and work towards it by gradually reducing the timer!

If you're playing from a computer, you might prefer using the left and right arrow keys instead of the black and white buttons on screen (you can also use enter to restart).
